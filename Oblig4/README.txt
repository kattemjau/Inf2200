Ukjente karakterer blir ikke registrert av program.
Liten test på starten av progammet.
Programmet kjores med: java main needle.txt haystack.txt
Kompileres med javac *.java
eller kjøres og kompileres med make for kun testing.

Programmet starter med å sette prioriteringer på needle. Dermed vet programmet hvor mange Characters den skal hoppe hver gang den får en bad char shift.
Sorterings algorithmen starter å sammenligne needelen bakfra, i starten av haystacken. Hvis det er en char i haystack som er ukjent i needlen så hopper needlen wildcard char bortover.
Ved at den vet hvor lang needelen er og hvilke bokstaver som er i needelen.
