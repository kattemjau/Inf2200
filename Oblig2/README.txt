Kjøring av fil kan gjøres med make

Makefilen vil automatisk kompilere filen og kjøre den

output.txt ble gennerert ved å kjøre
make FILE=buildhouse1.txt >> output.txt
med FILE som endret til de ulike filene.
>> operasjonen vil appende utskriften til output.txt filen

Oblig2.java er hovedfilen og blir kjørt av make

Alle class filene kan kjøres med make clean

Hele oppgaven burde fungere som den skal

All koden er skrevet inn i en fil. Hvis den er uoversiktelig kan koden åpnes i 2 faner hvor
man kan se på forskjellige kodedeler samtidig.
